ANOTACIONES SCRIPT MySQL

EL Script que se debe ejecutar es bd2_p1_mysql.sh

1)Dar permiso al script inicial para poder ejecutarlo
2)Cambiar de los scripts:

	bd2_p1_mysql.sh

	- Poner la contraseña de root que se tenga en ese dispositivo
	* Otra opción sería modificar el fichero mysqld.cnf, en el siguiente directorio añadiendo las siguientes sentencias: 
		[client]
		user = usuario
		password = poner_su_contraseña
		port = 3306
		host = localhost
	- Cambiar la dirección de la instrucción de chmod, en la línea 10, con el directorio donde se encuentren los scripts alojados
	

	cargar_ficheros_csv.sql

	- Modificar de todos los LOAD DATA LOCAL INFILE la dirección donde se encuentren los ficheros descargados,
	  será la dirección donde se encuentren los scripts más /ficheroscsv/nombre_fichero.csv
